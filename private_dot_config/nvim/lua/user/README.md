# Config for NeoVim ( AstroNvim )

## Install

Clear the old config:

```bash
rm -rf ~/.config/nvim
```

Install AstroNvim:

```bash
git clone --depth 1 https://github.com/AstroNvim/AstroNvim ~/.config/nvim
```

Apply the user config (Overwrite the default config):

```bash
chezmoi apply
```
